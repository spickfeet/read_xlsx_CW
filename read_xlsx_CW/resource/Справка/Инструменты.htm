Основные команды редактора

Запуск:

  • Запуск анализа: Пуск

Файловые операции:

  • Создать файл: Файл ⇒ Создать
  • Открыть файл: Файл ⇒ Открыть
  • Сохранить файл: Файл ⇒ Сохранить
  • Сохранить как: Файл ⇒ Сохранить как

Редактирование:

  • Отменить: Правка ⇒ Отменить
  • Вернуть: Правка ⇒ Вернуть
  • Копировать: Правка ⇒ Копировать
  • Вырезать: Правка ⇒ Вырезать
  • Вставить: Правка ⇒ Вставить
  • Удалить всё: Правка ⇒ Удалить всё

Информация:

  • Постановка задачи: Текст ⇒ Постановка задачи
  • Грамматика: Текст ⇒ Грамматика
  • Классификация грамматики: Текст ⇒ Классификация грамматики
  • Метод анализа: Текст ⇒ Метод анализа
  • Диагностика и нейтрализация ошибок: Текст ⇒ Диагностика и нейтрализация ошибок
  • Тестовый пример: Текст ⇒ Тестовый пример
  • Список литературы: Текст ⇒ Список литературы
  • Исходный код программы: Текст ⇒ Исходный код программы
