Информация о программе

Программа написана в рамках первой лабораторной работы по дисциплине "Теория формальных языков и компиляторов".

Программа доработана в рамках курсовой работы.

Техническое задание

Разработать приложение – текстовый редактор, дополненный функциями языкового процессора.

Приложение имеет графический интерфейс пользователя.

Язык реализации: C#.

Функциональные элементы

  1. Основное меню программы
    • Пункт меню Текст - при вызове команд этого меню открываются окна с соответствующей информацией по курсовой работе "Функция ввода Console.ReadLine() языка C#"
  2. Панель инструментов

    Содержит кнопки вызова часто используемых пунктов меню:

    • Файл - Создать, Открыть, Сохранить, Сохранить как, Выход
    • Правка - Отменить, Вернуть, Копировать, Вырезать, Вставить, Удалить всё
    • Пуск - команда для запуска анализатора текста
    • Вызов справки - запускает справочную систему с описанием всех реализованных функций меню
  3. Окно ввода/редактирования текста

    Область редактирования представляет текстовый редактор. Команды меню Файл и Правка работают с содержимым этой области.

  4. Окно отображения результатов

    Область для вывода сообщений и результатов работы языкового процессора. Ввод текста в этой области запрещен.
