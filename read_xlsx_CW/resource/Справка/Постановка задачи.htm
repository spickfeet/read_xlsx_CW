Ввод данных с консоли в C#

В языке C# для считывания пользовательского ввода с консоли используется метод Console.ReadLine(). Этот метод ожидает, пока пользователь введет строку и нажмёт Enter, после чего возвращает введённые данные в виде строки (string).

Console.ReadLine();

Синтаксический анализатор (парсер)

Парсер будет считать верными следующие записи:

"Console.ReadLine();"
"Console .ReadLine ();"
"Console . ReadLine ( ) ;"
